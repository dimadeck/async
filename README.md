#Asyncio, twisted, tornado#

Подготовка:

1) git clone https://github.com/dimadeck/async

2) cd async

3) python -m venv env

4) source env/bin/activate


##Twisted Simple Chat##

Запуск сервера:

python run.py


Подключение клиентов через telnet:

telnet 127.0.0.1 1234
